# 🚀 快速开始 - 算法可视化工具

## 最快速的使用方法 (30秒上手!)

### 方法1: 浏览器版本 (推荐 ⭐)

**无需安装任何东西!**

```bash
# 直接打开HTML文件
open algorithm_visualizer.html
```

或者直接双击 `algorithm_visualizer.html` 文件!

### 方法2: Python版本

```bash
# 1. 确保安装了matplotlib
pip install matplotlib

# 2. 运行程序
cd python
python algorithm_visualization.py
```

## 📖 3步使用教程

### 使用HTML版本:

1️⃣ **打开文件** → 双击 `algorithm_visualizer.html`

2️⃣ **选择算法** → 点击任意算法按钮(如"冒泡排序")

3️⃣ **开始演示** → 点击"▶️ 开始"按钮

**就是这么简单!** 🎉

## 🎮 控制说明

| 按钮 | 功能 |
|------|------|
| ▶️ 开始 | 开始/继续动画播放 |
| ⏸️ 暂停 | 暂停动画 |
| ⏭️ 下一步 | 单步执行,仔细观察 |
| 🔄 重置 | 回到初始状态 |
| 🎲 随机数据 | 生成随机测试数据 |

## 💡 使用技巧

### 技巧1: 调整速度
- 拖动速度滑块可以控制动画快慢
- 学习时建议调慢速度,仔细观察

### 技巧2: 自定义数据
- 在输入框中输入自己的数据
- 格式: `5, 2, 8, 1, 9, 3`(用逗号分隔)

### 技巧3: 单步调试
- 点击"下一步"按钮可以逐步执行
- 适合深入理解每一步操作

### 技巧4: 观察统计
- 注意右下角的统计信息
- 可以看到比较次数、交换次数等

## ❓ 常见问题

**Q: HTML文件打不开怎么办?**
- A: 右键点击文件 → 选择"打开方式" → 选择浏览器(Chrome/Firefox等)

**Q: Python版本报错怎么办?**
- A: 确保安装了matplotlib: `pip install matplotlib`

**Q: 动画太快/太慢?**
- A: 使用速度滑块调整动画速度

**Q: 如何保存动画?**
- A: 可以使用屏幕录制软件,或查看 VISUALIZATION_README.md 的高级用法

**Q: 可以添加自己的算法吗?**
- A: 可以!参考 VISUALIZATION_README.md 的贡献指南
