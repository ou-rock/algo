## 🎯 学习路径

### Level 1: 可视化入门 (初学者)

**目标**: 直观理解算法执行过程

**资源**:
- `algorithm_visualizer.html` - 浏览器打开即用
- 包含6种基础算法的动画演示

**涵盖算法**:
1. 冒泡排序 (Bubble Sort)
2. 插入排序 (Insertion Sort)
3. 选择排序 (Selection Sort)
4. 二分查找 (Binary Search)
5. 查找最大值 (Find Maximum)
6. 数组求和 (Array Sum)

**使用方法**:
```bash
# 直接在浏览器中打开
open algorithm_visualizer.html
```

## 💡 学习建议

### 对于初学者

1. **从可视化开始**
   - 先用HTML工具观察算法执行过程
   - 不要急于看代码
   - 理解"为什么这样做"

2. **循序渐进**
   - 从简单算法开始（求和、最大值）
   - 逐步过渡到排序算法
   - 最后学习高级算法

3. **动手实践**
   - 看懂≠会写
   - 必须自己实现一遍
   - 用Jupyter Notebook练习

### 对于有基础的学习者

1. **关注实际应用**
   - 理解算法在工程中的价值
   - 学习如何选择合适的算法
   - 考虑性能和权衡

2. **深入理解**
   - 分析时间复杂度
   - 理解空间复杂度
   - 掌握优化技巧

3. **举一反三**
   - 思考变种问题
   - 尝试优化实现
   - 应用到实际项目

### 对于面试准备

1. **刷题策略**
   - 先完成Notebook中的10道题
   - 掌握常见算法模板
   - 练习口述思路

2. **重点关注**
   - 动态规划（高频）
   - 图算法（Dijkstra、拓扑排序）
   - 字符串算法（KMP）
   - 经典数据结构（LRU、Trie）

3. **面试技巧**
   - 先分析问题
   - 讨论不同解法
   - 代码写清晰
   - 考虑边界情况

## 📈 性能对比

### 排序算法性能对比

| 算法 | 平均 | 最好 | 最坏 | 空间 | 稳定性 |
|------|------|------|------|------|--------|
| 冒泡排序 | O(n²) | O(n) | O(n²) | O(1) | 稳定 |
| 插入排序 | O(n²) | O(n) | O(n²) | O(1) | 稳定 |
| 选择排序 | O(n²) | O(n²) | O(n²) | O(1) | 不稳定 |
| 快速排序 | O(n log n) | O(n log n) | O(n²) | O(log n) | 不稳定 |
| 归并排序 | O(n log n) | O(n log n) | O(n log n) | O(n) | 稳定 |
| 堆排序 | O(n log n) | O(n log n) | O(n log n) | O(1) | 不稳定 |

### 查找算法性能对比

| 算法 | 平均 | 最好 | 最坏 | 空间 | 前提条件 |
|------|------|------|------|------|----------|
| 线性查找 | O(n) | O(1) | O(n) | O(1) | 无 |
| 二分查找 | O(log n) | O(1) | O(log n) | O(1) | 有序数组 |
| 哈希查找 | O(1) | O(1) | O(n) | O(n) | 哈希表 |
